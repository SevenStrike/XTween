### XTween 动画插件
#### 概述
XTween 是一个高性能的 Unity 动画系统，提供了丰富的动画类型、缓动效果和灵活的控制方式。它采用对象池技术优化性能，支持编辑器预览，适用于 UI 和游戏对象的动画处理。

#### 核心组件

##### XTween_Controller 动画控制器  
> 动画控制器，挂载在 GameObject 上用于创建和控制动画

- 支持多种动画类型（位置、旋转、缩放、颜色等）
- 可配置动画参数（持续时间、延迟、缓动模式等）
- 提供按键控制（播放、倒带、终止等）




#### XTween_Controller (动画控制器)
动画控制器，挂载在 GameObject 上用于创建和控制动画。

主要功能：

支持多种动画类型（位置、旋转、缩放、颜色等）

可配置动画参数（持续时间、延迟、缓动模式等）

提供按键控制（播放、倒带、终止等）
